# ⚖️ Legal Document Drafting Assistant

An AI-powered web application that generates legal documents such as complaints, RTI applications, and legal notices instantly using Generative AI.

---

## 🚀 Live Demo

🔗 https://legal-documents-drafting-ai-klmqt8jkhvyhibyxjcedpj.streamlit.app/

---

## 📌 Features

- ⚡ Instant legal document generation  
- 🌐 Supports Hindi & English  
- ⚖️ Multiple tones (Formal, Aggressive, Neutral)  
- 🧠 AI-powered drafting using LLM  
- 💻 Clean and user-friendly interface  

---

## 🛠 Tech Stack

- Python  
- Streamlit  
- OpenRouter API  
- Large Language Model (LLM)  

---

## ⚙️ How It Works

1. User selects document type  
2. User enters details  
3. AI processes the input  
4. Legal document is generated instantly
